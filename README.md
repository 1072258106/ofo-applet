## 本地运行
~~~
git clone https://github.com/MiceLiD/ofo-applet.git
~~~

微信开发者工具选择目录为ofo

## 非法域名报错
微信开发者工具中请关闭校验域名，或把www.easy-mock.com添加到域名配置中
